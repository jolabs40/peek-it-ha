# Peek-it [HA]

**Turn your Android TV into a smart notification display.**

Rich overlay notifications — cameras, dashboards, TTS, menus — on top of any running app, in real time.

- 🎨 **Design visually** — build your notifications by drag & drop in the in-app **Designer** (any browser). No code to write.
- 🏠 **Trigger from Home Assistant** — fire them from your automations in a few lines… or from Tasker, Node-RED, Jeedom, or any HTTP client.
- 📺 **Zero latency** — native Android overlay, no streaming, no casting.
- 🔔 Text, images, RTSP cameras, web pages, SVG, shapes, interactive buttons & menus, HA entity widgets, charts, text-to-speech.

## Quick start

1. Install the **Peek-it [TV]** app on your Android TV — [Google Play](https://play.google.com/store/apps/details?id=net.jolabs40.peekit)
2. Download this integration in HACS, then **restart Home Assistant**.
3. *Settings → Devices & Services → Add Integration → Peek-it [HA]*, and enter your TV's IP (default port `8081`).
